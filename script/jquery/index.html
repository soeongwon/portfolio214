제이쿼리

제이쿼리는 자바스크립트 라이브러리를 이용해 만든 언어입니다. 제이쿼리는 HTML에 포함되어 있는 클라이언트 스크립트 언어를 단순화하도록 설계된 웹브라우저 호환성을 가진 자바스크립트 라이브러리입니다. 라이브러리는 자주 사용하는 코드를 재사용 및 효율성 있게 사용할 수 있도록 만들어진 함수의 집합을 의미합니다.

제이쿼리 언어의 특징

  • 제이쿼리는 CSS를 쉽게 적용할 수 있습니다.
  • 제이쿼리는 크로스브라우징을 지원합니다.
  • 제이쿼리는 플러그인이 풍부합니다.
  • 제이쿼리는 코드를 적게, 효율적으로 작성할 수 있습니다.
  • 제이쿼리는 좋은 확장성과 Ajax 기능을 구현합니다.

선택자

기본 선택자

종류 예시 설명
태그 선택자 $("div") 기본 태그를 선택
클래스 선택자 $(".div") 클래스 태그를 선택
아이디 선택자 $("#div") 아이디 태그를 선택
그룹 선택자 $("#div .div") 여러가지 태그를 선택
전체 선택자 $("*") 전체 태그를 선택

계층 선택자

종류 예시 설명
하위 선택자 $("div p") div 태그의 모든 자식 중 p태그를 모두 선택
자식 선택자 $("div > p") div 태그의 모든 자식 중 첫 번째 자식 p태그를 선택
형제 선택자 $("div + p") div 태그의 인접한 형제 p태그 중 첫번째 태그만 선택
형제들 선택자 $("div ~ p") div 태그의 인접한 형제 p태그를 모두 선택

속성 선택자

종류 예시 설명
[name |="value"] $("li a[href |='web']") a태그 속성 중 'web'과 같은 속성값을 선택
[name *="value"] $("li a[href *='web']") a태그 속성 중 'web'이 포함되어 있는 같은 태그를 선택
[name~="value"] $("li a[href~='web']") a태그 속성 중 'web'으로 시작되는 태그를 선택
[name $="value"] $("li a[href $='web']") a태그 속성 중 'web'으로 끝나는 태그를 선택
[name !="value"] $("li a[href !='web']") a태그 속성 중 'web'이 포함되지 않는 태그를 선택
[name ^="value"] $("li a[href ^='web']") a태그 속성 중 'web'으로 시작하는 태그를 선택
[name] $("li a[href]") a태그 속성 중 일치하는 태그를 선택
[name="value"][name="value"] $("li a[href][class]") a태그 속성 중 일치하는 태그를 선택

메서드 설명
.children() 선택한 요소의 바로 아래 자식 요소만 선택합니다.

.find() ** 선택한 요소의 자식 요소 중 조건에 맞는 요소를 선택합니다.

.parent() 선택한 요소의 부모 요소를 선택합니다.
.parents() 선택한 요소의 모든 부모 요소를 선택합니다.
.parentsUntil() 선택한 요소의 부모 요소 중 조건에 맞을 때까지 찾습니다.
.prev() 선택한 요소의 이전 요소를 선택합니다.
.prevAll() 선택한 요소의 이전 모든 요소를 선택합니다.
.prevUntil() 선택한 요소의 이전 요소 중 조건에 맞을 때까지 찾습니다.
.siblings() 선택한 요소의 형제 요소를 모두 찾습니다.

$("selector").stop(clearQueue, goToEnd)
clearQueue : 큐에 대기중인 효과들을 삭제할 것인지 여부 설정
goToEnd : 진행중인 애니메이션을 완료할 것인지를 결정

.stop(true, true) : 일단 현재 진행되고 있는 효과는 정지되면서 효과의 맨 마지막으로 이동한 후 큐에 남아있는 효과는 모두 지우고 끝낸다
.stop(true, false) : 일단 현재 진행되고 있는 효과는 즉시 그 지점에서 정지되고 큐에 남아있는 효과는 모두 지우고 끝낸다
.stop(false, true) : 일단 현재 진행되고 있는 효과는 정지면서 맨 마지막으로 즉시 이동한 후 큐에 남아 있는 효과를 수행한다
.stop(false, false) : 일단 현재 진행되고 있는 효과는 즉시 그 지점에서 정지되고 큐에 남아있는 효과를 수행한다
